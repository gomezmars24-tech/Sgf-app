🔥 Activer Firebase — 1 étape
📋 Copie ce lien dans ton navigateur :
https://console.firebase.google.com/project/sgf-database/firestore/rules
Sur la page qui s'ouvre :
1. Tu vois un éditeur de texte
2. Sélectionne tout le texte (Ctrl+A)
3. Supprime et colle exactement ceci :
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
4. Clique "Publier"
5. Reviens ici et clique "Tester la connexion"
